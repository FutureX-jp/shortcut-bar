# ショートカットバー — 使い方

デスクトップに常駐する、フローティング型のランチャーです。
アプリ/URLのショートカット、起動中アプリの一覧、時計、クリップボード履歴、
ウィンドウのライブプレビュー（覗き窓）などをまとめた「小さなタスクバー」。
着せ替えテーマで、レトロ携帯ゲーム機風の十字メニュー画面にもできます。

---

## 動かし方

### exe版を受け取った人（Python不要・追加インストール不要）
1. `ShortcutBar.exe` をダブルクリックするだけで起動します。必要な実行ライブラリは
   exeに同梱済みなので、別途インストールは要りません。
2. 初回に Windows SmartScreen の警告（「WindowsによってPCが保護されました」）が
   出ることがあります。作者が個人のため署名が無いだけで、動作に問題はありません。
   **「詳細情報」→「実行」** で起動できます。
3. 終了は、左上の赤い信号機ボタン、または右クリック →「✕ 終了」。

### ソース版を受け取った人（Python が入っている場合）
- `pythonw shortcut_bar.py` で起動（コンソール無し）。
- 初回に `Pillow` と `pystray` が自動でインストールされます
  （トレイ／アイコン表示に使用。失敗してもその機能だけ無効化されて動きます）。

---

## 最初にやること
- 右クリック → **⚙ 設定…** で、表示するセクションや外観を調整できます。
- 右クリック → **＋ ショートカットを追加** で、よく使うアプリ/URLを登録。
- 上部の帯をドラッグすると位置を移動、右下の `⠿` を左右にドラッグで横幅を変更。

---

## 主な機能（標準UI）
- **検索バー**: 入力でショートカット絞り込み、Enterで起動／URL直打ち／Google検索。
  さらに **電卓にもなります**：`3980*1.1` のように式を打つと `= 4378` と表示され、
  Enterで答えをコピーできます。
- **起動中アプリ一覧**: クリックで前面化／再クリックで最小化。各行の 📷 でスクショ。
  右クリックで 最大化／最小化／閉じる、行を**ドラッグして放り投げる**と左半分・
  右半分・最大化に配置（窓投げ）。
- **覗き窓（ライブプレビュー）**: 起動中アプリや「最近のウィンドウ」の行にマウスを
  乗せると中身を小さくライブ表示（設定でON/OFF）。「窓の透視」で対象以外を薄くも可。
- **▦ Exposé**: 全ウィンドウのライブサムネイルを並べてクリック切替。
- **📺 PiP監視**: 行の右クリック →「ミニ表示で監視」で、その窓を常時ミニ表示。
- **重力モード / 付箋マグネット / ボスが来た**: ツールメニューの遊び・便利機能。
- **クリップボード履歴・最近のウィンドウ・セクション折りたたみ**。
- **自動起動**: 設定 → システム → スタートアップに登録。

## 着せ替えテーマ
設定 → 外観 で、Apple風のほか **Windows XP / 7 / 8 / 10 / 11 /
レトロ携帯ゲーム機風** に一発切替（再起動不要）。

### レトロ携帯ゲーム機風テーマ（十字メニュー）
月替わりカラー・波背景・十字メニューの、ゲーム機のホーム画面風UIです。
- **横断検索**: 画面上部の「文字入力で検索」のとおり、**キーボードで文字を打つだけ**で
  ショートカット・ゲーム・仕事・カスタム列・起動中アプリを横断で絞り込み。
  ↑↓で選択・Enterで開く・Escで閉じる（ツール列の「検索」からも開けます）。
- **お気に入りピン**: 項目を右クリック（△）→「先頭に固定」で、その列の先頭に固定。
- **ゲーム列**: インストール済みの **Steam / Epic Games** のゲームが自動で並びます
  （決定で起動。実機風のボックスアート表示。設定→本体設定でON/OFF）。
- **仕事モード列**: 図面・現場フォルダ・CADなど、仕事で使うファイル/フォルダ/URLを
  まとめる専用列。列の末尾「＋ファイル／＋フォルダ／＋URL」で登録（設定でON/OFF）。
- **カスタムカテゴリ**: 設定 →「カテゴリを追加…」で、自分専用の列を作れます
  （無料版は1つまで／Pro版は無制限）。
- **写真/ミュージック/ビデオ**: 「フォルダを追加…」で自分のフォルダを登録、決定で
  中身を展開。写真は選択すると背景に大きくプレビュー。
- **操作**: 右クリック=オプション（△）、中クリック=戻る（×）、しばらく操作しないと
  タイトルバーだけに自動折りたたみ（秒数は設定・0=OFF）。決定後は開いた窓へ
  カーソルが自動移動。設定列に**システム情報**（OS/CPU/メモリ/稼働時間）。

## Pro版について
無料版でもほぼ全機能が使えます。**Pro版**では今のところ
**カスタムカテゴリを無制限に作成**できます（無料版は1つまで／今後Pro専用機能を追加予定）。
プロダクトキーをお持ちの場合は、設定 →「ライセンス / Pro版」から
購入時のメールアドレスとキーを入力すると有効になります。

---

## 使用状況の共有（テレメトリ）について
アプリ改善のため、**同意した場合のみ**、匿名の使用状況を開発者に送る機能があります。
- 初回起動時に「送ってもいいですか？」と一度だけ確認します。**いいえでもOK**、
  すべての機能はそのまま使えます。
- 送る内容は **バージョン・OSの種類・使った機能・エラー情報・匿名ID** のみ。
  **ファイル名・URL・パスなど個人を特定する情報は送りません**。
- あとから 設定 →「使用状況の共有」でいつでもON/OFFを切り替えられます。

## 自動アップデート
新しいバージョンが公開されると、起動時に「更新しますか？」と聞いてきます。
「はい」でダウンロード→置き換え→再起動まで自動です（設定 → システム でOFF可）。

## 設定の保存先
`%USERPROFILE%\.shortcut_bar.json`（初回起動時に自動生成）。
この1ファイルに設定・登録ショートカットが保存されます。削除すると初期状態に戻ります。

## 動作環境
- Windows 10 / 11（64bit）
- 覗き窓プレビューはデスクトップ合成（DWM）を使用します（標準で有効）。

## アンインストール
- exe（と `screenshots` フォルダ）を削除するだけ。
- 自動起動を登録していた場合は、設定 → システム →「削除」で解除。
- 設定を完全に消すなら `%USERPROFILE%\.shortcut_bar.json` を削除。

## うまく起動しないとき
- 「python…dll をロードできません」等が出る場合は、古いバージョンの可能性があります。
  最新版（実行ライブラリ同梱）を入れ直してください:
  https://github.com/FutureX-jp/shortcut-bar/releases/latest/download/ShortcutBar.exe
