# ショートカットバー — 使い方（配布用）

デスクトップの右端などに常駐する、フローティング型のランチャーです。
アプリ/URLのショートカット、起動中アプリの一覧、時計、クリップボード履歴、
ウィンドウのライブプレビュー（覗き窓）などをまとめた「小さなタスクバー」です。

---

## 動かし方

### exe版を受け取った人（Python不要）
1. `ショートカットバー.exe` をダブルクリックするだけで起動します。
2. 初回に Windows SmartScreen の警告（「WindowsによってPCが保護されました」）が
   出ることがあります。作者が個人のため署名が無いだけで、動作に問題はありません。
   **「詳細情報」→「実行」** で起動できます。
3. 終了は、左上の赤い信号機ボタン、または右クリック →「✕ 終了」。

### ソース版を受け取った人（Python が入っている場合）
- `pythonw shortcut_bar.py` で起動（コンソール無し）。
- 初回に `Pillow` と `pystray` が自動でインストールされます
  （トレイ／アイコン表示に使用。失敗してもその機能だけ無効化されて動きます）。

---

## 最初にやること
- 右クリック → **⚙ 設定…** で、表示するセクションや外観を調整できます。
- 右クリック → **＋ ショートカットを追加** で、よく使うアプリ/URLを登録。
- ヘッダー（上部の帯）をドラッグすると位置を移動できます。
- 右下の `⠿` を左右にドラッグで横幅を変えられます。

## 主な機能
- **検索バー**: 入力でショートカット絞り込み、Enterで起動／URL直打ち／Google検索。
  さらに **電卓にもなります**：`3980*1.1` のように式を打つと `= 4378` と表示され、
  Enterで答えをコピーできます。
- **起動中アプリ一覧**: クリックで前面化／再クリックで最小化。各行の 📷 でスクショ。
  - **右クリック**で 最大化／最小化／元に戻す／閉じる などのメニュー。
  - **行をドラッグして放り投げる**と、離した場所でウィンドウを
    左半分・右半分・最大化に配置できます（窓投げ）。
- **覗き窓（ライブプレビュー）**: 起動中アプリや「最近のウィンドウ」の行にマウスを
  乗せると、その中身が小さくライブ表示されます（設定でON/OFF可）。
  - オプション「**窓の透視**」をONにすると、ホバー中はその窓以外が薄くなり
    場所がひと目で分かります。
- **▦ Exposé（全ウィンドウ一覧）**: ヘッダーの ▦ で全ウィンドウのライブ
  サムネイルが並び、クリックで切り替え。Escか何もない所クリックで閉じる。
- **📺 PiP監視**: 行の右クリック →「ミニ表示で監視」で、その窓を小さな
  ライブ画面として常時表示。上部バーの − ＋ でサイズ、✕ で閉じる。
  ダブルクリックで本体ウィンドウへ。
- **着せ替えテーマ**: 設定 → 外観 で Apple風のほか
  **Windows XP / 7 / 8 / 10 / 11 / レトロ携帯ゲーム機風**に一発切替（再起動不要）。
  レトロ携帯ゲーム機風テーマは月替わりカラー・波背景・十字メニューつき。
  - 写真/ミュージック/ビデオ列の「**フォルダを追加…**」で自分のフォルダを登録可能。
  - しばらく操作しないと**タイトルバーだけに自動で折りたたみ**（マウスを乗せると復帰）。
  - 項目を決定すると開いた窓へ**カーソルが自動で移動**します（どちらも設定列でOFF可）。
- **自動アップデート**: 新しいバージョンが公開されると、起動時に
  「更新しますか？」と聞いてきます。「はい」でダウンロード→置き換え→
  再起動まで自動です（設定 → システム でOFFにもできます）。
- **目玉**: ヘッダーの目玉がマウスを追いかけます。**クリックするとツール
  メニュー**（下記の遊び機能など）が開きます。
- **重力モード**: ツールメニュー → 🌍 で、ショートカットのアイコンが
  落下・バウンド。ドラッグで投げて遊べます（アイコンはクリックで起動も可能）。
- **付箋マグネット**: ツールメニュー → 🧲 で、散らばった付箋をバーの横に整列。
- **最近のウィンドウ**: 直前まで使っていたウィンドウにワンクリックで戻れます。
- **クリップボード履歴**: 直近コピーをクリックで再コピー。
- **セクション折りたたみ**: 各見出しの ▽ / ▷ をクリックで開閉。
- **ボスが来た**: 遊び系ウィンドウを一斉に隠して仕事用を前面に。
- **自動起動**: 設定 → システム → スタートアップに登録。

## 設定の保存先
`%USERPROFILE%\.shortcut_bar.json`（初回起動時に自動生成）。
この1ファイルに設定・登録ショートカットが保存されます。削除すると初期状態に戻ります。

## 動作環境
- Windows 10 / 11
- 覗き窓プレビューはデスクトップ合成（DWM）を使用します。標準で有効です。

## アンインストール
- exe版: exe（と `screenshots` フォルダ）を削除するだけ。
- 自動起動を登録していた場合は、設定 → システム →「削除」で解除してください。
- 設定を完全に消すなら `%USERPROFILE%\.shortcut_bar.json` を削除。

## 自動アップデートの配り方（配布する人向け）

受け取る人は何もしなくてOK。配布する側は以下を1回だけ準備する。

1. **exeの置き場所を用意**（無料ならGitHubのReleasesが定番）:
   - GitHubにリポジトリを作る（コード非公開ならexeだけ上げる空リポでも可）
   - Releases → New release → exeを添付してPublish
   - exeのURLは毎回固定にできる:
     `https://github.com/<user>/<repo>/releases/latest/download/<exe名>`
2. **マニフェストJSONを置く**（同じリポジトリに `update.json` をコミットが楽）:
   ```json
   {"version": "1.1.0",
    "url": "https://github.com/<user>/<repo>/releases/latest/download/ShortcutBar.exe",
    "notes": "更新内容がダイアログに表示されます",
    "sha256": ""}
   ```
   URLは `https://raw.githubusercontent.com/<user>/<repo>/main/update.json`
3. **アプリ側にURLを設定**: 設定 → システム → アップデートのURL欄に
   マニフェストのURLを入れる。**この設定込みでexeを渡せば**、受け取った人は
   以後ずっと自動更新される（設定は各自の `.shortcut_bar.json` に入る）。

新バージョンを出すときは: `sbar/version.py` のAPP_VERSIONを上げる →
`build_exe.bat` → Releasesにexeを上げ直す → `update.json` のversionを上げる。
それだけで配布済み全員に更新が届く。
