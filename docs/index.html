使い方ガイド

できることが多いので、まずはここだけ。あとは触りながら覚えられます

1

起動する

ダウンロードした ShortcutBar.exe をダブルクリックするだけ。インストール不要で、画面の端に小さなバーが常駐します。

2

ショートカットを追加

バーを右クリック →「📌ショートカットを追加」。「🔍アプリから探す」に名前を打つと候補が出るので、選ぶだけで名前・アイコンが自動で入ります。URLやフォルダも登録OK。

3

自分好みに着せ替え

⚙設定 → テーマ でワンクリック着せ替え（再起動不要）。配置 →「画面端に固定」にすると、Windowsのタスクバー代わりに使えます。

🔎横断検索・電卓

バー上部の検索欄に文字を打つと、アプリ・URLをまとめて検索。12*8 のように式を打てばそのまま電卓になります。

👁覗き窓・PiP監視 一部Pro

起動中アプリの行にカーソルを合わせると中身をプレビュー。右クリックからミニ表示（PiP）で、変換や解析の完了を見張れます。

📷ウィンドウをスクショ

起動中アプリの行の 📷 を押すと、その窓だけを画像として保存できます。資料づくりや記録に便利。

🔑シークレットメモ Pro

右クリック → 🚀ツール → 🔑。パスワードなどを暗号化して保存し、クリックでコピー → 30秒後に自動消去。このPCの本人だけが復号できます。

🌤天気・防災パネル

右クリック → 🚀ツール → 🌤天気。今の気温・湿度・風に加え、24時間の降水確率グラフ・週間予報・警報・最近の地震をひと目で。地域は都道府県 → 市区町村を選ぶだけ（入力不要）。雨が近づくとお供がひとこと教えてくれます。

✎スタートメニューを自分仕様に

クラシック（青）のスタートメニューは「✎スタートを編集」で項目のオン/オフと並び替えが自由。ヘッダーの🔍に打てばツール・ショートカット・アプリを横断してそのまま起動できます。

🐱デスクトップのお供

小さな相棒が住みつきます。ドラッグで移動、ダブルクリックでなでなで、ファイルをドラッグすればゴミ箱へ。ときどき便利な使い方も教えてくれます。

🎨壁紙モード Pro

スマホのホーム画面のように、好きな画像を1枚まるごとバーの背景に。文字色・書体もカスタムでき、明るさに応じて文字が自動で読みやすく調整されます。

迷ったら、まずバーを右クリック。使える機能がメニューにまとまっています。

よくある質問

本当に無料で使えますか？

はい。標準機能はすべて無料で、期限もありません。レトロ携帯機風テーマのみ、無料版では5日間の体験期間があります（ミニマル・クラシックはずっと無料）。Pro版は壁紙モードなど一部の追加機能を解放するものです。

インストールは必要ですか？

不要です。exeをダウンロードしてダブルクリックするだけ。必要な実行ライブラリは同梱済みです。

SmartScreenの警告が出ます

個人開発のため署名が無いだけで動作に問題はありません。「詳細情報」→「実行」で起動できます。

おすすめの使い方はありますか？

作者のおすすめは、Windowsの標準タスクバーを隠してショートカットバーをタスクバー代わりにする使い方です。設定は2ステップ:

  1. バーの ⚙設定 → 配置 → 「画面端に固定」 で好きな画面端にドッキング
  2. Windowsの設定 → 個人用設定 → タスクバー → 「タスクバーを自動的に隠す」をON

アプリの切替・時計・クリップボード履歴はバーに揃っているので、標準タスクバーなしで画面を広く使えます。

PCを買い替えたらキーは？

同じメールアドレスとキーでそのまま再認証できます。回数制限はありません。

支払い方法は？

各種クレジットカード・Apple Payに対応（Stripeで安全に決済されます）。PayPayは現在審査中で、近日対応予定です。
