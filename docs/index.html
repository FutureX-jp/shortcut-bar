よくある質問

Macやスマホで使えますか？

いいえ、本アプリはWindows 10 / 11専用です。Mac・Linux・スマートフォン（iPhone / Android）では動作しません。ご購入前にお使いのPCがWindowsであることをご確認ください。

本当に無料で使えますか？

はい。標準機能はすべて無料で、期限もありません。天気・雨雲レーダーも無料です。レトロ携帯機風テーマのみ、無料版では5日間の体験期間があります（ミニマル・クラシックはずっと無料）。Pro版は壁紙モードなど一部の追加機能を解放するものです。

インストールは必要ですか？

不要です。exeをダウンロードしてダブルクリックするだけ。必要な実行ライブラリは同梱済みです。

SmartScreenの警告が出ます

個人開発のため署名が無いだけで動作に問題はありません。「詳細情報」→「実行」で起動できます。

おすすめの使い方はありますか？

作者のおすすめは、Windowsの標準タスクバーを隠してショートカットバーをタスクバー代わりにする使い方です。設定は2ステップ:

  1. バーの ⚙設定 → 配置 → 「画面端に固定」 で好きな画面端にドッキング
  2. Windowsの設定 → 個人用設定 → タスクバー → 「タスクバーを自動的に隠す」をON

アプリの切替・時計・天気・クリップボード履歴はバーに揃っているので、標準タスクバーなしで画面を広く使えます。

天気のデータはどこから？

日本国内は気象庁の公開データ（予報・警報・地震・雨雲ナウキャスト）を使用し、雨雲レーダーの地図は国土地理院のタイルを使用しています。海外の地点はOpen-Meteoを利用します。

PCを買い替えたらキーは？

同じメールアドレスとキーでそのまま再認証できます。回数制限はありません。
