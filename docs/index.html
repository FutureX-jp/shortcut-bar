よくある質問

本当に無料で使えますか？

はい。標準機能はすべて無料です。Pro版は一部の追加機能を解放するものです。

インストールは必要ですか？

不要です。exeをダウンロードしてダブルクリックするだけ。必要な実行ライブラリは同梱済みです。

SmartScreenの警告が出ます

個人開発のため署名が無いだけで動作に問題はありません。「詳細情報」→「実行」で起動できます。

おすすめの使い方はありますか？

作者のおすすめは「タスクバー置き換え」スタイルです。①バーの設定（⚙）→「画面端に固定」で好きな画面端にドッキング → ②Windowsの設定 → 個人用設定 → タスクバー → 「タスクバーを自動的に隠す」をON。標準タスクバーの代わりにショートカットバー1本で運用でき、画面が広く使えます（起動中アプリの切替・時計・クリップボード履歴はバー側に揃っています）。

PCを買い替えたらキーは？

同じメールアドレスとキーでそのまま再認証できます。回数制限はありません。

支払い方法は？

各種クレジットカード・Apple Payに対応（Stripeで安全に決済されます）。PayPayは現在審査中で、近日対応予定です。
